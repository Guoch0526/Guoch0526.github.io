gulp 常用插件学习总结

发表于 2017-04-23 | 分类于 自动化构建工具


1、匹配符 、*、！、{}

2、文件操作

del (替代gulp-clean)

gulp-rename(重命名文件)

gulp-concat(合并文件)

gulp-filter(在虚拟文件流中过滤文件)

3、压缩

gulp-uglify(压缩js)

gulp-csso(压缩优化css)

gulp-html-minify(压缩html)

gulp-imagemin(压缩图片)

gulp-zip(zip压缩文件)

4、JS/CSS自动注入

gulp-autoprefixer(自动为css添加浏览器前缀)

gulp-rev(给静态资源文件名添加hash值)

gulp-rev-replace(重写被gulp-rev重命名的文件名)

5、工具

gulp-sass(编译sass)

gulp-babel(将ES6代码编译成ES5)

node.js 实现简易爬虫程序
